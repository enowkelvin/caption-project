# capestone-project
# Objective:
The objective is to analyze the different is service usage by subscribers and causal users
and patterns that support data-driven decision-making.

Analysing the ways different customers utilise bike services
This project is intended to know the different in usage between casual users and subscribers
in order to know if casual users could be converted to subscribers. This capstone porject will 
be done using R program to prepare datasets, combine them into one single dataset,
then cleaning, analysing and visualisation.
